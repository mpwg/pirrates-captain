# Product Vision

Pirrates Captain acts as the command bridge for the media automation stack.

Instead of opening multiple web interfaces, users manage everything from a single native iOS app.

## Goals
- Simplify management of Sonarr/Radarr
- Provide a unified dashboard
- Make media automation accessible on mobile

## Core Features

### Dashboard
- System health
- Active downloads
- Recent additions
- Upcoming episodes

### Discover
Global search for:
- Movies
- TV shows
- Music

### Library
Browse media managed by arr services.

### Activity
Monitor download queues and history.

### Automation (future)
Rules such as:
Download all new Marvel movies.
Upgrade quality automatically.
